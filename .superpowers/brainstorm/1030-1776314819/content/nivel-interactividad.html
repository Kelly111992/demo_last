¿Qué nivel de interactividad quieres en las demos?

Esto define cuánto código y lógica nueva agregaremos a cada demo.

A

Enriquecida — Prompts + UI mejorada

Mejores system prompts, más quick replies contextuales, respuestas más ricas con datos estructurados, mejor diseño visual. El chat sigue siendo el motor principal.


Ejemplos: respuestas con tablas de comparación, sugerencias dinámicas según contexto, diseño más pulido por industria.

B

Flujos reales — Estado + formularios + acciones

Cada demo tiene flujos completos: carrito de pedidos, calendario de citas, formularios de datos, confirmaciones, historial. La IA guía flujos complejos con lógica de negocio real.


Ejemplos: restaurante con carrito funcional, gimnasio con inscripción paso a paso, clínica que recoge datos del paciente y confirma cita.

C

Híbrido — Flujos completos en las mejores demos

Las 3-4 demos más impactantes (restaurante, clínica, gimnasio, hotel) reciben flujos completos de negocio. Las demás reciben mejoras de prompts y UI.


Máximo impacto visual con esfuerzo focalizado. Las demos de showcase quedan impresionantes.
